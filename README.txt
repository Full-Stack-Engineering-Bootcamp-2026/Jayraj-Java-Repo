Project: Student Info API

Description:
This is a Spring Boot application that provides student information using REST APIs.

Features:

* /students → returns list of students
* /students/{id} → returns student by ID
* /students?batch=CDAC-2026 → filter students by batch
* /batch → returns batch name from configuration

How to Run:

1. Open terminal in project folder
2. Run: ./mvnw spring-boot:run
3. Access APIs at http://localhost:8082

Technologies:

* Java
* Spring Boot
