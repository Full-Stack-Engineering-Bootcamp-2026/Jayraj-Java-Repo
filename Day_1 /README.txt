Project: Hello World Spring Boot APIProject: Hello World Spring Boot API

Description:
This is a simple Spring Boot application with basic REST endpoints.

Features:

* /hello → returns greeting
* /about → returns bio
* /status → checks if application is running
* /time → shows current server time (bonus)
* /about-json → returns JSON response (bonus)

How to Run:

1. Open terminal in project folder
2. Run: ./mvnw spring-boot:run
3. Open browser and test endpoints on:
   http://localhost:8081

Technologies Used:

* Java
* Spring Boot
